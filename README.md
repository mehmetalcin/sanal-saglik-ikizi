# Sanal Sağlık İkizi (Virtual Health Twin) - v5.0 Mega Pack 🚀🦾

Bu proje, geleneksel sağlık takip uygulamalarını **Yüksek Çözünürlüklü 3D Humanoid Avatar Engine** ve **Proaktif Sağlık Koçu** ile modernize eden gelişmiş bir web uygulamasıdır.

## ✨ Öne Çıkan Özellikler

- **Humanoid Avatar Engine (Avatar 5.0):** 
  - 20'den fazla özelleştirilebilir skin (Karakter kütüphanesi).
  - VKİ (BMI) verilerine göre anlık şekillenen dinamik vücut yapısı.
  - Mimikler (Göz kırpma, gülümseme) ve duygu durum entegrasyonu.
- **Proaktif Sağlık Koçu:**
  - Günün ilerleyişine (Fractional Day Progress) göre su ve adım önerileri.
  - Sesli/Balon uyarılı interaktif geri bildirim.
- **Akıllı Veri Yönetimi:**
  - "Smart Fetching" ile sekmeler arası anlık geçiş ve veri önbellekleme.
  - Granüler veri silme (Tek bir öğünü silerken günün geri kalanını korur).
- **Gelişmiş Tahlil Takibi:**
  - PDF tahlil yükleme ve otomatik veri çıkarımı.
  - Kritik değerlerin trend grafikleri üzerinden takibi.

## 🛠️ Teknoloji Yığını
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Backend:** PHP 7.4+, MySQL/PDO.
- **Grafikler:** Chart.js, SVG Animation Engine.

## 🚀 Kurulum
1. Dosyaları web sunucunuza (Apache/Nginx) yükleyin.
2. `api/db.php` dosyasındaki veritabanı bilgilerini güncelleyin.
3. Tarayıcınızdan `index.html` dosyasını çalıştırın.
